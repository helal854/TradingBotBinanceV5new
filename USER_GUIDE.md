## 🎯 الإشارات

### أنواع الإشارات
البوت يوفر إشارات تداول متنوعة:
- **إشارات الشراء** (LONG) 📈
- **إشارات البيع** (SHORT) 📉
- **إشارات التحديث** (تعديل المراكز) ♻️
- **إشارات الإغلاق** (إغلاق المراكز) ✅

### معلومات الإشارة
كل إشارة تحتوي على:
- **العملة**: مثل BTCUSDT, ETHUSDT
- **الاتجاه**: شراء أو بيع
- **سعر الدخول**: السعر المقترح للدخول
- **الأهداف**: مستويات الربح المتوقعة
- **وقف الخسارة**: مستوى الحماية من الخسائر
- **الرافعة المالية**: مضاعف التداول المقترح

### مثال على إشارة
```
🎯 إشارة جديدة

💰 العملة: BTCUSDT
📈 الاتجاه: شراء (LONG)
💵 سعر الدخول: 45,000
🎯 الهدف الأول: 46,000
🎯 الهدف الثاني: 47,000
🛑 وقف الخسارة: 44,000
⚡ الرافعة: 10x
```
